Acqua Gold Impermeabilizações - pacote do site

Estrutura:
- index.html
- assets/css/style.css
- assets/js/main.js
- assets/icons/sprite.svg
- assets/images/*

Como usar:
1. Extraia o arquivo ZIP.
2. Abra o index.html no navegador.
3. Se quiser publicar, envie toda a pasta para a hospedagem.

Observações:
- O WhatsApp e os telefones já estão configurados no layout.
- O mapa pode ser ajustado depois com o endereço exato da empresa.
